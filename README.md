# Mihir Thumbnail Portfolio

A professional portfolio website for **Mihir Rath**, a YouTube Thumbnail Strategist focused on increasing **CTR (Click-Through Rate)** through data-driven thumbnail design.

The website showcases thumbnail case studies, performance results, and a strategic design workflow aimed at helping creators get more views.

---

## Live Website

View the live portfolio here:

https://mihir-thumbnail-portfolio.vercel.app

---

## Features

- Hero section introducing thumbnail strategy approach
- Concept-to-final thumbnail breakdown
- Real case studies with CTR improvements
- Organized thumbnail gallery by niche
- Strategy workflow explanation
- Client testimonials
- Direct contact via WhatsApp

---

## Tech Stack

- **React**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **shadcn/ui**
